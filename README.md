# CRM Cafe

Web application CRM untuk cafe dengan fokus pada loyalty program, promo, reward, member, dan integrasi POS.

## Ringkasan

CRM Cafe membantu pelanggan mengumpulkan poin dari setiap transaksi, menukar reward, dan mendapatkan informasi promo terbaru. Aplikasi ini dirancang mobile-friendly dan terhubung dengan sistem POS internal untuk sinkronisasi transaksi dan poin.

## Fitur MVP

- Registrasi dan login pelanggan
- Riwayat transaksi dan poin
- Penukaran reward
- Informasi promo terbaru
- Dashboard admin untuk mengelola promo, reward, dan member
- Integrasi POS untuk sinkronisasi transaksi dan poin

## Tech Stack

- Frontend: Next.js
- Backend: Laravel
- Database: PostgreSQL
